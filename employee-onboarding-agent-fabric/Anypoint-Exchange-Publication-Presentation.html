MCP Services Suite

Four interconnected microservices providing comprehensive employee onboarding automation

Employee Onboarding MCP

Core employee database operations with PostgreSQL/H2 fallback support and intelligent error handling.

  • Multi-database support (PostgreSQL/H2)
  • CRUD operations with validation
  • Status tracking and management
  • Performance monitoring
  • RESTful API endpoints

Agent Broker MCP

Intelligent process orchestration hub managing workflows and coordinating service interactions.

  • Workflow orchestration
  • Service coordination
  • Event-driven architecture
  • Process monitoring
  • Agent network management

Asset Allocation MCP

Comprehensive IT asset management for laptops, mobile phones, ID cards, and equipment tracking.

  • Asset inventory management
  • Allocation workflows
  • Asset tracking & history
  • Availability management
  • Maintenance scheduling

Notification MCP

Advanced email notification system with Gmail SMTP integration and rich HTML templating.

  • Gmail SMTP integration
  • Rich HTML templates
  • Automated notifications
  • Delivery tracking
  • Template customization
