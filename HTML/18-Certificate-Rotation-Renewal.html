9. Troubleshooting Certificate Issues

Common Certificate Problems

Diagnose and resolve certificate-related issues

Common Certificate Issues:
  • Certificate Expired: Certificate has passed its validity period
  • Hostname Mismatch: Certificate doesn't match the requested hostname
  • Untrusted CA: Certificate authority not in trust store
  • Chain Incomplete: Missing intermediate certificates
  • Private Key Mismatch: Certificate and private key don't match
# Certificate Debugging Commands # Check certificate details openssl x509 -in certificate.crt -text -noout # Verify certificate chain openssl verify -CAfile ca-bundle.crt certificate.crt # Test TLS connection openssl s_client -connect api.company.com:443 -servername api.company.com # Check certificate expiration openssl x509 -in certificate.crt -noout -enddate # Verify certificate and private key match openssl x509 -noout -modulus -in certificate.crt | openssl md5 openssl rsa -noout -modulus -in private.key | openssl md5 # Test certificate with specific protocol curl -v --cert client.crt --key client.key https://api.company.com/health
